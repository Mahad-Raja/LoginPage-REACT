# 🔐 React Login System with Protected Routes
A beginner-friendly React project demonstrating how to build a Login System using **React Router** and **Protected Routes (Navigation Guard)**.
This project was created to learn the fundamentals of React Router, navigation, routing, and route protection.
## 🚀 Features
- Login Page
- Landing Page
- React Router Navigation
- Protected Route (Navigation Guard)
- Programmatic Navigation using `useNavigate`
- Login Validation
- Logout Functionality
- Modern UI with Tailwind CSS
- Responsive Login Card
## 🛠️ Technologies Used
- React
- React Router DOM
- Tailwind CSS
- JavaScript (ES6+)
- Vite
